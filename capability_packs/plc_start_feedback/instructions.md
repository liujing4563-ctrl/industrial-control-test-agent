# PLC 电机启动反馈测试 — 操作说明

## 测试目标

验证 PLC I/O 系统中电机启动反馈回路是否正常：
1. 互锁检查 — 确认安全条件满足
2. 启动信号输出 — 通过 PLC 发出启动指令
3. 反馈信号等待 — 监控电机启动反馈
4. 环境复位 — 测试完成后恢复初始状态

## 测试流程

1. 读取 PLC 互锁信号 → 确认急停/安全门/热保护状态
2. 发送测试启动信号 (write_test_signal)
3. 等待电机启动反馈信号
4. 记录反馈时序
5. 复位测试环境

## 故障场景

| 故障 | 症状 | 可能原因 |
|------|------|---------|
| 互锁阻塞 | read_interlock 返回 all_clear=false | 急停按下/安全门打开/热保护触发 |
| 启动无反馈 | wait_feedback 超时 | 电机故障/接触器不吸合/反馈线路断开 |
| 信号映射错误 | 读到非预期的信号值 | PLC 地址映射配置错误 |
| 环境无法复位 | reset_test_environment 失败 | 设备状态不一致/残留输出 |

## 前置条件

- PLC 已上电并处于 STOP 或 TEST 模式
- 所有互锁条件正常
- 测试环境与生产环境隔离
- 电机处于可安全启动状态
