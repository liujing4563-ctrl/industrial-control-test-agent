# 系统上下文

## 外部角色

- 测试工程师：创建并验证测试案件，审批设备修改。
- 验证工程师：分析故障、批准变更、确认回归结果。
- 测试负责人：监控案件进度与评估结果。
- MCU 测试台：提供 MCU 连接与测试接口。
- PLC 测试台：提供 PLC I/O 控制与信号采集。
- 模拟器：提供硬件替代运行环境与模拟信号。
- MCP 服务：暴露测试工具与设备能力。
- LLM 提供商：提供模型推理能力（仅作为运行时组件依赖）。
- GitHub/测试管理系统：提供代码、工单和结果管理接口。

## 系统边界

本平台负责：

- 案件管理与状态保持。
- 多智能体与能力包调度。
- ActionIntent 生成与安全校验。
- 设备操作请求的审批与执行流程。
- Evidence 记录与治理。

外部服务负责：

- MCU/PLC/模拟器的具体工具执行。
- 现场安全、急停、硬实时保障。
- 真实 LLM 调用与供应商密钥管理。
- 外部工单与测试管理系统的数据同步。
